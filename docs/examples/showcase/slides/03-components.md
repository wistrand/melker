# Components

Melker ships with a rich set of built-in components:

| Component    | Purpose                    |
|--------------|----------------------------|
| `data-table` | Sortable, selectable table |
| `markdown`   | Rich text with GFM         |
| `canvas`     | Pixel graphics and shaders |
| `dialog`     | Modal overlays             |
| `tabs`       | Tabbed interfaces          |
| `split-pane` | Resizable panels           |

---

All components support **CSS styling**, **theming**, and **accessibility**.
