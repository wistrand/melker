# Basics

Learning progression for Melker beginners. Start with hello and work your way up.

| File                          | Concepts                                                    |
|-------------------------------|-------------------------------------------------------------|
| `hello.melker`                | Minimal app, text, button, event handler                    |
| `counter.melker`              | State, getElementById, setValue, style tag                  |
| `form-demo.melker`            | Input fields, form layout                                   |
| `dialog-demo.melker`          | Modal dialogs, show/hide                                    |
| `tabs-demo.melker`            | Tabbed interface, tab switching                             |
| `text-styles.melker`          | Font weight, style, decoration, colors                      |
| `overflow.melker`             | Scrollable containers, overflow modes                       |
| `spinner.melker`              | Spinner component, animation                                |
| `toast.melker`                | Toast notifications                                         |
| `opacity.melker`              | Opacity and background opacity                              |
| `css-variables.melker`        | CSS custom properties (variables)                           |
| `css-combinators.melker`      | CSS selector combinators (child, descendant, sibling)       |
| `css-nesting.melker`          | CSS nesting syntax                                          |
| `pseudo-classes.melker`       | CSS pseudo-classes (:hover, :focus, :active)                |
| `transitions.melker`          | CSS transitions and animations                              |
| `animation.melker`            | Keyframe animations                                         |
| `palette-demo.melker`         | Command palette integration                                 |
| `before-exit.melker`          | Exit confirmation handler                                   |
| `command-test.melker`         | Command element and keyboard shortcuts                      |
| `state-binding-count.melker`  | createState, bind, boolean CSS classes                      |
| `state-binding.melker`        | State binding with sync function, bind-mode="one-way"       |
| `two-way-binding.melker`      | Two-way binding, checkbox toggle visibility, bind-mode      |

## Running

```bash
./melker.ts examples/basics/hello.melker
./melker.ts examples/basics/counter.melker
```
