# Basics

Learning progression for Melker beginners. Start with hello and work your way up.

| File                 | Concepts                                   |
|----------------------|--------------------------------------------|
| `hello.melker`       | Minimal app, text, button, event handler   |
| `counter.melker`     | State, getElementById, setValue, style tag |
| `form-demo.melker`   | Input fields, form layout                  |
| `dialog-demo.melker` | Modal dialogs, show/hide                   |
| `tabs-demo.melker`   | Tabbed interface, tab switching            |

## Running

```bash
./melker.ts examples/basics/hello.melker
./melker.ts examples/basics/counter.melker
```
