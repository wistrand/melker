# Components

Component demos - one canonical example per component type.

## Running

```bash
./melker.ts examples/components/input.melker
./melker.ts examples/components/slider.melker
```

## Input Components

| File | Description |
|------|-------------|
| `input.melker` | Single-line text input with Enter key handling |
| `textarea.melker` | Multi-line text input |
| `slider.melker` | Numeric slider with range and step |
| `progress.melker` | Progress bar display |

## Selection Components

| File | Description |
|------|-------------|
| `select.melker` | Dropdown selection |
| `combobox.melker` | Editable dropdown with filtering |
| `autocomplete.melker` | Text input with suggestions |
| `command-palette.melker` | Keyboard-driven command picker (Ctrl+K style) |

## Data Display

| File | Description |
|------|-------------|
| `table.melker` | Basic table with sorting and selection |
| `data-table.melker` | Data-driven table with computed columns |
| `data-bars.melker` | Bar charts (horizontal, vertical, sparklines) |
| `segment-display.melker` | 7-segment LED-style numeric display |

## Navigation

| File | Description |
|------|-------------|
| `file-browser.melker` | File system browser with keyboard navigation |
