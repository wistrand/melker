# Layout Examples

Flexbox layout, borders, and scrolling demonstrations.

| File                          | Description                              |
|-------------------------------|------------------------------------------|
| `flex-demo.melker`            | Comprehensive flexbox layout examples    |
| `flexbox-visualizer.melker`   | Interactive flexbox property explorer    |
| `borders.melker`              | Border styles and box model              |
| `border-title.melker`         | Borders with title text                  |
| `one-column-scroll.melker`    | Single column scrolling                  |
| `three-column-scroll.melker`  | Multi-column scrolling layout            |
| `horizontal-column-scroll.melker` | Horizontal scroll layout             |

## Running

```bash
./melker.ts examples/layout/flex-demo.melker
./melker.ts examples/layout/flexbox-visualizer.melker
```
