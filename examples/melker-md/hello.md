# Hello World

A minimal melker-block example.

## Layout

```melker-block
+--App---------------------------------+
| style: border: thin; padding: 1      |
| +--Greeting------------------------+ |
| | type: text                       | |
| | text: Hello from ASCII boxes!    | |
| | style: color: green              | |
| +----------------------------------+ |
+--------------------------------------+
```

## Notes

This is the simplest possible melker-block example:
- Single `melker-block` with nested text element
- No scripts, styles, or components needed
- Properties defined inline in the ASCII box
